## mgrote.tor-node

### Beschreibung
Setzt ein tor-relay ODER eine [tor-bridge](https://community.torproject.org/relay/setup/bridge/debian-ubuntu/) auf.
ORPort muss in Firewall freigeschaltet sein.
Es muss eine Portfreigabe im Router existieren.

### getestet auf
- [x] Ubuntu (>=20.04)
- [ ] Debian
- [ ] ProxMox 6.1
