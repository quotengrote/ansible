## mgrote.acng
### Beschreibung
Installiert Apt-Cacher-NG.
Die Daten werden auf der Maschine abgelegt.

### Funktioniert auf
- [x] Ubuntu (>=18.04)
- [ ] Debian


### Variablen + Defaults
##### Auf welchen Port ACNG laufen soll
- acng_server_port: "9999"
##### Wie lange Pakete gespeichert werden sollen
- acng_exthreshold: "60"
